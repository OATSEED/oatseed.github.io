เกี่ยวกับโครงการ

"คลังเกมการศึกษาอ้างอิงหลักสูตรแกนกลาง" ช่วยให้ครู ผู้ปกครอง และนักเรียน ค้นหาสื่อเกมออนไลน์ที่สอดคล้องกับตัวชี้วัดของหลักสูตรแกนกลางได้ง่ายและรวดเร็ว ข้อมูลเก็บบนฐานข้อมูล Supabase แก้ไขได้จริงผ่านเว็บโดยแอดมินที่ล็อกอินด้วยบัญชี Google ที่ได้รับอนุญาตเท่านั้น

อ้างอิงหลักสูตรจริง

ทุกเกมระบุรหัสตัวชี้วัดและสมรรถนะสำคัญ พร้อมข้อความอ้างอิงสำหรับแนบในแผนการสอน

ฐานข้อมูลจริง เรียลไทม์

ข้อมูลเก็บบน Postgres ผ่าน Supabase ปลอดภัยด้วย Row Level Security และล็อกอินจริงผ่าน Google

ฟรี ไม่มีค่าใช้จ่าย

เข้าถึงคลังเกมและสื่อประกอบได้ฟรี เพื่อสนับสนุนการจัดการเรียนรู้ในทุกโรงเรียน
